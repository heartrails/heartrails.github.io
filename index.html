理念
----
つくり、磨き続ける。

ハートレイルズは新しい価値を常に模索し、つくり、磨き続けます。
ハートレイルズ自らがその理念を貫き、また、つくる人を支援することで、
世の中に様々な価値を提供し、社会を豊かにします。


目標
----
世界で最も信頼される開発会社になる。

ハートレイルズは速く、上手く、良いサービスをつくることで信頼を積み重ね、
その信頼を糧にさらに良いサービスをつくります。


行動指針
----
- プロとしての自負と誇りを持って働く
- HRT （謙虚、尊敬、信頼） を忘れない
- 他責でなく自責を前提に考える
- 失敗を次の成功に活かす
- 変化を過度に恐れない
- よく学び、よく遊ぶ


開発指針
----
- 自律的に協働する
- 手段に固執せず目的を重視する
- QCD のバランスを意識する
- 定型業務は自動化する
- DRY、KISS、YAGNI、コードレビュー
- 情報共有、状況の見える化にこだわる


採用指針
----
- なるべく多くのスタッフとの面談、技術試験 （2 週間 ～） を経るようにする
- 一緒に気持ち良く働けそうかどうかを重視する
- 細かい技巧よりも本質的な素養を持っているかどうかを重視する
- 一度不採用となった人、一度退職した人の再応募をそれ以外の人と区別しない


ツール
----
- Slack （コミュニケーション）
- Google Apps （会議、ドキュメント、カレンダー）
- GitHub （コード管理、タスク管理）
- Jenkins、CircleCI （デプロイ管理）
- Ruby on Rails、koa （フレームワーク）
- Elixir、Go （注力言語）
- AWS、GCP （インフラ）


カルチャー
----
- リモートワークを前提にする
　（異なる場所、異なる時間で働くことを許容する）
- 振り返りミーティング
　（個人、会社の KPT を定期的に話し合う）
- 勉強会
　（持ち回りで各自の学びを共有する）
- リモート部活動
　（料理部、筋肉部等）
- プライベートを大事にする
　（業務時間外の飲み会、休日の行事を催さない）
